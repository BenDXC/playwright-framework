# Playwright Test Framework
![Playwright Tests](https://github.com/<BenDXC>/playwright-framework/actions/workflows/playwright.yml/badge.svg)

## 🚀 Features

* Page Object Model (POM)
* Data-driven testing
* API + UI validation
* Auth reuse with storageState
* Parallel execution

## ▶️ Run Tests

```bash
npm install
npx playwright install
npx playwright test
```

## 📊 Reports

```bash
npx playwright show-report
```

## 🧠 Design

* Tests → specs
* UI → pages
* API → api layer
* Setup → fixtures
